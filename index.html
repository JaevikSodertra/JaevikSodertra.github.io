Стриминг CN→RU: статусы грузов и SLA

Источники: китайские TMS/WMS (webhooks + периодический polling). Нормализация справочников, дедупликация по composite-key, маршрутизация в топики Kafka. Оркестрация в Airflow, слой staging → marts в ClickHouse.

  • Снижение «слепых зон» по статусам ~70%
  • Алерты по SLA: −18% штрафов
  • Событие появлялось в аналитике через минуты

Калькулятор полной логистической стоимости

dbt-модели с тестами и версионированием. Витрина учитывает пошлины, СВХ, брокераж, утилизацию, доп-работы (оклейка, дооснащение). Единые правила расчёта + журнал изменений.

  • Ошибки в отчётности −40%
  • Время расчёта — секунды

Дивидендная аналитика (RU рынок)

ETL календарей/прайсов, нормализация МСФО/РСБУ, расчёт DY, payout ratio, сезонности. Дашборды и автопостинг сводок.

  • Airflow, Pandas, Parquet, Metabase
  • Телеграм-отчёты по расписанию

Наблюдаемость и качество данных

SLI/SLA, freshness/uniqueness, пороговые алерты, trace-логирование. Бейзлайны по сезонности для снижения ложных тревог.

  • Great Expectations / Soda, Prometheus + Grafana
  • MTTR ↓ ~35%
